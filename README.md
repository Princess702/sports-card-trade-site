# CardSwap Dugout

A professional, static sports card trading marketplace prototype with searchable card-style listings, a trade cart, card-photo upload requests, and an admin review dashboard.

## Run locally

```bash
npm run dev
```

Then open <http://localhost:5173>.

## Validate

```bash
npm run build
```

Check that the clean PR contains no unresolved merge conflict markers:

```bash
npm run check:conflicts
```
